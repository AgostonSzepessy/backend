# Installing
1. Create the database using `create_db.sql`.
2. `node-gyp` must be installed globally, along with a C++ compiler (required for argon2). The
compiler must either be
    * GCC >= 4.8
    * Clang >= 3.3
    * Visual Studio 2015 or newer
3. Run `npm install`

# Running
```
npm run dev
```

# Transpile to JS
```
npm run tsc
```
